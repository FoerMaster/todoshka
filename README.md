Тудушка
=============
Простое веб-приложение для учета ведения дел.

Стек
==========
Webpack, ReactJS, TypeScript, AntD, SASS, NodeJS.

## Как запустить?

Запуск проекта в dev режиме
```cmd
npm run start
```

Упаковка проекта
```cmd
npm run build
```

Запуск webpack serve функции
```cmd
npm run serve
```

Проверить код при помощи linter'a
```cmd
npm run lint
```
